# 🤖 Intelligent Document Processing Platform (IDPP) 📄

## 📋 Overview

An AI-powered document processing solution that automates classification, extraction, and search for large volumes of documents. Built for a major client.


## ✨ Features

- 🔍 **Vector-Based Search**: Intelligent search with context-aware results
- 🏷️ **AI-Powered Classification**: Automatic categorization of 50+ document types
- 📊 **Data Extraction**: Smart extraction of key-value pairs from documents
- 📁 **Document Management**: Organized storage with consistent naming conventions
- 🔒 **User Access Control**: Role-based permissions with maker-checker approval
- 📈 **Confidence Scoring**: Reliability metrics for extracted information
- 🔄 **Pipeline Optimization**: Multiprocessing for enhanced performance

## 🛠️ Technologies

- **Backend**: Python, Azure Functions
- **AI/ML**: GPT Models, Azure Document Intelligence
- **Prompt Engineering** - Extensive exhaustive prompt writing and instruction to GPT 
- **Storage**: Azure Blob Storage
- **Search**: Azure AI Search
- **Deployment**: Azure App Service
- **DevOps**: GitHub Actions, CI/CD Pipelines
- **Authentication**: Azure Active Directory

## 📊 Performance

- 📈 **Classification Accuracy**: 85% across 54 document types
- ⚡ **Processing Speed**: Optimized for files up to 10MB
- 🔍 **Search Relevance**: Top 30 results based on vector similarity
- 🔄 **Throughput**: Multiprocessing for parallel document handling

## 🔧 Key Components

### 1. Search Module

- Vector-based document matching
- Keyword navigation within documents
- Class-based filtering of results

### 2. Document Management System

- CIF-based organization
- Classification of documents into predefined types
- Confidence scores for extraction accuracy
- Editable extracted data

### 3. User Management

- Maker-Checker approach for changes
- Role-based access control
- Activity logging for audit purposes

## 💡 Innovations

- **Prompt Engineering**: Custom prompts for document classification and extraction
- **JSON Data Structuring**: Organized extraction of key-value pairs
- **Multiprocessing Implementation**: Enhanced pipeline performance
- **PDF Chunking**: Optimized thread utility for large documents
